PHP

Les variables

PHP est un language permissif

Vous n'avez pas besoin de spécifier le type de variable

Vous pouvez aussi changer de type de variable à la volée

Exemple


<?php
    $foo = 'bar';
    echo $foo;

    $foo = 1;
    echo $foo;

Déclaration

Contrairement au JS, il n'est pas nécessaire d'indiquer que l'on créé une variable

Les variables classiques commencent par un $

Déclaration

Elles ne peuvent pas commencer par un chiffre

Elles ne peuvent pas contenir d'espace ni de caractères spéciaux (@,é,^,-,...)

Déclaration

Comme en JS, il vous faut choisir une nomenclature

  • camelCase : variables classiques
  • underscore_case : variables classiques
  • PascalCase : Classes
  • UPPER_CASE : Constantes

Types

Il existe différents types de variables en PHP

  • String (string)
  • Int (int)
  • Float (float)
  • Array (array)
  • Boolean (bool)
  • Object (object)
  • Function
  • etc.

Types


<?php
    $foo = 'bar';              // String (fonctionne avec "")
    $foo = 12;                 // Int
    $foo = 1.2;                // Float
    $foo = ['blabla', 'toto']; // Array
    $foo = false;              // Boolean
    $foo = new stdClass();     // Object (?)

Types

Il est possible de connaître le type d'une variable avec la fonction gettype()


<?php
    $foo = 'bar';
    echo gettype($foo); // Affichera "string"

Types

Il est possible de changer le type d'une variable


<?php
    $foo = '12';      // String
    $foo = (int)$foo; // Int

Tableaux

Déclaration


<?php
    $foo = [];

    // Ancienne façon
    $foo = array();

Tableaux

Un tableau peut stocker n'importe quel type de variable (y compris un autre tableau)


<?php
    $foo = ['bar', 12, ['deuxième', 'tableau']];

Tableaux

Affichage


<?php
    $foo = ['bar', 12, ['deuxième', 'tableau']];
    
    echo $foo[0];
    echo $foo[1];
    echo $foo[2][0];
    echo $foo[2][1];

Objets

Les objets sont un peu plus difficiles à créer

Il faut instancer une classe StdClass et accéder à ses propriétés avec ->


<?php
    $foo = new StdClass();
    $foo->toto = 'tata';
    
    echo $foo->toto;

Concaténation

Il est possible de concaténer des variables avec le .


<?php
    $toto = 'foo';
    $tata = 'bar';
    $tutu = $toto.' '.$tata;

    echo 'result: '.$tutu;

Il est aussi possible d'injecter des variable directement dans une chaîne de caractères avec les double quotes "


<?php
    $toto = 'foo';
    $tata = 'bar';

    echo "result: $toto $tata";

⚠️ N'utilisez les " que si vous souhaitez injecter des variables car elles demandent plus de ressource

Opérations

Il est possible d'effectuer des opérations


<?php
    $foo = 1;
    $bar = 2;
    $result = $foo + $bar;

    echo $result;

☝️ Contrairement au JS, le + ne permet de faire que des additions

Variables globales

Il est possible de créer des variables impossible à modifier et qui seront visibles de n'importe où à partir du moment où elles sont déclarées

Pour cela on utilise la fonction define()


<?php
    define('FOO', 'bar');
    echo FOO;

On utilise souvent du SCREAMING_SNAKE_CASE pour ce genre de variables

Elles sont en général utilisées pour des variables de configuration qui ne devraient pas changer et dont on peut avoir besoin n'importe où dans le code
