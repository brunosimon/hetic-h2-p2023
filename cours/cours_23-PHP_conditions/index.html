PHP

Conditions

IF

IF

If permet de tester qu'une condition soit vraie

Fonctionne comme en javascript


<?php
    $foo = true;

    if($foo == true)
    {
        echo 'Vraie !';
    }

Vraie ! va s'afficher car $foo = true

IF

Lorsqu'il n'y a qu'une seule instruction, il n'est pas nécessaire d'écrire les accolades { }


<?php
    $foo = true;

    if($foo == true)
        echo 'Vraie !';

    // 'Vraie !' va s'afficher

IF

Si l'on teste que la valeur doit être true ou false il n'est pas nécessaire de l'écrire


<?php
    if($foo == true)
        echo 'Vraie !';

    if($foo == false)
        echo 'Faux !';

Équivaut à


<?php

    if($foo)
        echo 'Vraie !';

    if(!$foo)
        echo 'Faux !';

IF


<?php
    $foo = 1;

    if($foo == '1')
        echo 'Vraie !';

Va renvoyer Vraie ! car, avec ==, PHP va comparer la variable sans distinction de type

IF


<?php
    $foo = 1;

    if($foo === true)
        echo 'Vraie !';

Ne va pas renvoyer Vraie ! car, avec === (triple égalité), PHP va aussi tester le type de la variable

IF

Exercice


<?php
    if('1' == 1)
        echo 'vraie !';

    if('1' === 1)
        echo 'vraie !';
        
    if('1' !== 1)
        echo 'vraie !';
        
    if('1' != 1)
        echo 'vraie !';
        
    if('1' == true)
        echo 'vraie !';
        
    if(1 === true)
        echo 'vraie !';

IF

Avec plus petit que et plus grand que


<?php
    $foo = 20;

    if($foo < 30)
        echo 'Vraie !';

    if($foo > 10)
        echo 'Vraie !';

    if($foo <= 20)
        echo 'Vraie !';

    if($foo >= 20)
        echo 'Vraie !';

AND

Nous souhaitons tester si l'utilisateur peut acheter de l'alcool en fonction de son âge et si le magasin est ouvert


<?php
    $age = 18;
    $storeOpen = true;

    if($age >= 18)
    {
        if($storeOpen)
        {
            echo 'Can buy alcool !';
        }
    }

Cette syntaxe fonctionne, mais nous aimerions le faire en un seul if

AND


<?php
    if($age >= 18 AND $storeOpen)
    {
        echo 'Can buy alcool !';
    }

En PHP, le AND s'écrit en général &&


    if($age >= 18 && $storeOpen)

OR

Nous arrivons à un feu tricolor et nous souhaiterions savoir si nous pouvons avancer


<?php
    $color = 'green';

    if($color == 'green')
    {
        echo 'Fonce !';
    }

    if($color == 'orange')
    {
        echo 'Fonce !';
    }

Cette syntaxe fonctionne, mais nous aimerions le faire en un seul if

OR


<?php
    $color = 'green';

    if($color == 'green' OR $color == 'orange')
    {
        echo 'Fonce !';
    }

En PHP, le OR s'écrit en général ||


    if($color == 'green' || $color == 'orange')

AND / OR

Le AND et le OR peuvent être combinés


<?php
    $country = 'france';
    $age = 20;
    $storeOpen = true;

    if((($country == 'france' && $age >= 18) || ($country == 'usa' && $age >= 21)) && $storeOpen)
    {
        echo 'Can buy alcool !';
    }

Utilisez les parenthèses pour regrouper les tests

Évitez de faire des conditions trop complexes

ELSE

Comme en JS, le ELSE est interprété si le IF ne l'est pas


<?php
    $foo = false;

    if($foo)
    {
        echo 'dans le IF';
    }
    else
    {
        echo 'dans le ELSE';
    }

ELSE

On peut rajouter une condition au else avec else if


<?php
    $age = 2;

    if($age < 3)
    {
        echo 'Bébé';
    }
    else if($age < 10)
    {
        echo 'Gamin';
    }
    else if($age < 18)
    {
        echo 'Jeune';
    }
    else if($age < 30)
    {
        echo 'Adulte';
    }

SWITCH ... CASE

Switch ... Case

Le format suivant peut-être écrit plus simplement avec un switch


<?php
    $color = 'green';

    if($color == 'green')
    {
        echo 'Avance';
    }
    else if($color == 'orange')
    {
        echo 'Fonce';
    }
    else if($color == 'red')
    {
        echo 'Freine';
    }
    else
    {
        echo 'Fais gaffe';
    }

Switch ... Case

Avec le switch


<?php
    $color = 'green';

    switch($color)
    {
        case 'green':
            echo 'Avance';
            break;

        case 'orange':
            echo 'Fonce';
            break;

        case 'red':
            echo 'Freine';
            break;
            
        default:
            echo 'Fais gaffe';
            break;
    }

IF ternaire

Il est possible de faire un IF directement dans une assignation


<?php
    $age = 20;

    if($age >= 18)
        $status = 'adult';
    else
        $status = 'child';

Devient


<?php
    $age    = 20;
    $status = $age >= 18 ? 'adult' : 'child';

IF ternaire

Mais aussi dans un echo


<?php
    $age = 20;

    if($age >= 18)
        echo 'adult';
    else
        echo 'child';

Devient


<?php
    $age = 20;
    
    echo $age >= 18 ? 'adult' : 'child';
