PHP

Fonctions

Une fonction exécute des instructions

Les accolades { } sont obligatoires


<?php
    function helloWorld()
    {
        echo 'Hello World';
    }

    helloWorld();

La fonction peut renvoyer une valeur


<?php
    function helloWorld()
    {
        return 'Hello World';
    }

    echo helloWorld();

Et recevoir des paramètres séparés par virgules ,


<?php
    function multiply($a, $b)
    {
        return $a * $b;
    }

    echo multiply(10, 20);

Il est possible de spécifier des paramètres par défault

Et donc de ne pas les renseigner durant l'appel


<?php
    function multiply($a = 10, $b = 20)
    {
        return $a * $b;
    }

    echo multiply();

Scope PHP

Les variables déclarées en dehors de la fonction n'existent pas à l'intérieur

Il faut utiliser l'instruction global suivie de la variable pour y avoir accès dans la fonction


<?php
    $c = 30;

    function multiply($a = 10, $b = 20)
    {
        global $c;
        return $a * $b * $c;
    }

    echo multiply();

Les variables globales (créées avec define) ignorent le scope et sont disponibles partout


<?php
    define('C', 30);

    function multiply($a = 10, $b = 20)
    {
        return $a * $b * C;
    }

    echo multiply();
